# Política de Privacidade - TechFix

**Última atualização:** 13 de Abril de 2026

## 1. DADOS COLETADOS
Coletamos os seguintes dados para funcionamento da plataforma:
- **Identificação:** Nome, E-mail, Telefone, CPF/CNPJ.
- **Geolocalização:** Utilizada para mostrar serviços próximos ao técnico.
- **Mídia:** Fotos e vídeos carregados nos chamados e perfis (Estante da Credibilidade).

## 2. FINALIDADE
Os dados são usados exclusivamente para:
1. Validar a identidade dos usuários.
2. Processar pagamentos via Stripe/Mercado Pago.
3. Facilitar a comunicação via WhatsApp e E-mail.

## 3. SEGURANÇA
- Utilizamos criptografia SSL em todas as transações.
- Senhas são armazenadas com hashing de alta segurança (bcrypt).
- Não vendemos dados para parceiros de marketing.

## 4. SEUS DIREITOS
Você pode, a qualquer momento:
- Solicitar a exclusão total da sua conta.
- Exportar seus dados cadastrais.
- Editar suas preferências de notificação.
